### Projet des établissements universitaires algériens
Ce projet présente une base de données des enseignants des universités algériennes, construite en utilisant des fichiers binaires avec des index. Les fichiers sont structurés comme des tableaux de blocs ordonnés de taille fixe. Les index sont structurés comme des fichiers bitmap. Le choix de ces structures permet d'exécuter différentes opérations sur les données de manière rapide et efficace.
